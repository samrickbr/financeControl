# Finance Control API 🚀

Sistema de controle financeiro pessoal desenvolvido com **Java 17** e **Spring Boot**.

## 🛠️ Tecnologias
* Java 17 / Spring Boot
* Spring Data JPA
* PostgreSQL (Banco de Dados)
* Postman (Testes de API)

## 📌 Endpoints Principais
* `POST /lancamentos`: Registra uma nova despesa ou receita.
* `GET /usuarios/{id}`: Busca detalhes de um usuário.

## 📝 Notas de Versão
- Persistência funcionando no PostgreSQL.
- Retorno de JSON estruturado com relacionamento de tabelas.~~~~
